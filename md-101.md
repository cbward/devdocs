# Headers

# Header 1 - Syntax: \# Header 1

### Header 3 - Syntax: \### Header 3

##### Header 5 - Syntax: \##### Header 5
###### Header 6 - Syntax: \###### Header 6

# Links
To create a link, follow this pattern:

[Google](https://www.google.co.uk) - Syntax: \[Google](https://www.google.co.uk)
